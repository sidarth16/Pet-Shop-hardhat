# **Pet-Shop-hardhat**
Hardhat Implementation of https://trufflesuite.com/guides/pet-shop<br/>

**Try running some of the following tasks:**
<br/><br/>

- To Deploy Contract in localhost:
    ```shell
    npx hardhat node
    npx hardhat run scripts/deploy.js --network localhost
    ```

- To Run hardhat test : 
    ```shell
    npx hardhat test
    ```
